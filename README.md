Detecting Nature In Pictures
===============

Building
---------------

1. Add Dataset to the `/data` folder

2. Build with `./build.sh`

3. Run with `./DetectingNature`


Dataset Structure
---------------

	data/
		dataset_name/
			class_name/
				image_name.jpg
				another_image.png
				...
			another_class/
				...
		another_dataset/
			...

		
Example datasets
---------------
